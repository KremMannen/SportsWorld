SportsWorld API

Simple index of available endpoints and example requests

Base URL: /api

Available Resources

Athletes

Manage athletes

/athlete

Finances

Financial records

/finance

Venues

Sports venues

/venue

Example Requests

  • GET http://localhost:5110/api/athlete
  • GET http://localhost:5110/api/athlete/5
  • POST http://localhost:5110/api/finance

Endpoints — Athletes

  • GET /api/athlete List all athletes
  • GET /api/athlete/{id} Get athlete by id
  • GET /api/athlete/byname/{name} Get athletes by name
  • POST /api/athlete Create athlete object
  • POST /api/imageupload/athlete Upload image of athlete
  • PUT /api/athlete Update athlete
  • DELETE /api/athlete/{id} Delete by id

Endpoints — Finances

  • GET /api/finance List all finances
  • POST /api/finance Create finance object
  • PUT /api/finance Update finance
  • DELETE /api/finance Delete all finance data

Endpoints — Venues

  • GET /api/venue List all venues
  • GET /api/venue/{id} Get venue by id
  • GET /api/venue/byname/{name} Get venues by name
  • POST /api/venue Create venue object
  • POST /api/imageupload/venue Upload image of venue
  • PUT /api/venue Update venue
  • DELETE /api/venue/{id} Delete by id
